# ⚙️ Installation

## 📱 Termux / NetHunter

```bash
pkg update -y
pkg install python ffmpeg -y
pip install yt-dlp colorama
termux-setup-storage
python yt_downloader.py
```
## 💻 kali Linux installation 

```bash
sudo apt update
sudo apt install python3-pip ffmpeg -y
pip install yt-dlp colorama
python3 yt_downloader.py
```
